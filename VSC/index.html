Hipocykloida

Hipocykloida to krzywa płaska, która tworzy punkt na obwodzie mniejszego okręgu toczącego się bez poślizgu wewnątrz większego okręgu

\[ \begin{aligned} x(\theta) &= (R - r) \cos(\theta) + r \cos\left(\frac{R - r}{r} \theta\right) \\ y(\theta) &= (R - r) \sin(\theta) - r \sin\left(\frac{R - r}{r} \theta\right) \end{aligned} \]

Python

    import numpy as np
    import matplotlib.pyplot as plt
    
    # Parametry
    R = 5  # Promień większego okręgu
    r = 3  # Promień mniejszego okręgu
    
    # Zakres kąta
    theta = np.linspace(0, 2 * np.pi, 1000)
    
    # Równania hipocykloidy
    x = (R - r) * np.cos(theta) + r * np.cos((R - r) / r * theta)
    y = (R - r) * np.sin(theta) - r * np.sin((R - r) / r * theta)
    
    # Rysowanie
    plt.figure(figsize=(6, 6))
    plt.plot(x, y)
    plt.title("Hipocykloida")
    plt.xlabel("x")
    plt.ylabel("y")
    plt.axis('equal')
    plt.grid(True)
    plt.show()
